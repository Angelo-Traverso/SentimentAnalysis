-----------------------------------------------------------
Download and Run
-----------------------------------------------------------
- Download the folder "AngeloTraverso_ST10081927_PDAN8412_Part1"
- Unzip the folder
- Open Visual Studio > File > Open Folder/ Choose Folder > "AngeloTraverso_ST10081927_PDAN8412_Part1"
- You can then choose to Run All or just run individual cells using the 'play' button next to each cell
-----------------------------------------------------------
Libaries
-----------------------------------------------------------
Before running, ensure the necessary libraries are installed:
- tensorflow:   pip install tensorflow
- pyspark:      pip install pyspark
- seaborn:      pip install seaborn
- scikit-learn: pip install scikit-learn
- nltk:         pip install nltk
